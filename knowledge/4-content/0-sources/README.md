# 0-sources — Bài DSC cũ cần Refresh / Repurpose

Folder này chứa các bài viết **đã xuất bản của DSC** cần được viết lại, cập nhật, hoặc chuyển sang format mới.

## Phân biệt với các folder khác

| Folder | Nội dung | Khi nào dùng |
|---|---|---|
| `0-sources/` | Bài DSC cũ cần refresh | Khi bài đã outdate hoặc cần nâng chất |
| `1-outlines/` | Outline mới đã research | Sau khi chạy `/outlining` |
| `2-drafts/` | Bản nháp đang viết | Trong quá trình `/drafting` |
| `3-finalized/` | Bài đã xuất bản | Sau khi `/approve` |

## Quy trình dùng file trong folder này

1. Đặt bài DSC cũ vào đây (copy từ CMS hoặc website)
2. Chạy `/optimize [path]` để re-optimize bài cũ, **hoặc**
3. Chạy `/outlining [keyword]` → agent sẽ đọc file tương ứng ở đây làm tham khảo baseline

## Naming convention

Giữ nguyên tên file gốc (title của bài) để dễ nhận biết. Ví dụ:
```
ADX là gì_ Cách sử dụng chỉ báo ADX trong giao dịch.md
```
