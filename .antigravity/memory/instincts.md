# Continuous Learning: Instincts (Bản năng)

File này lưu trữ những "bản năng" mà hệ thống AI học được qua quá trình sửa bài cùng người dùng. 
Mỗi khi người dùng đưa ra phản hồi chỉnh sửa (VD: "bạn viết mở bài quá sáo rỗng"), hãy trích xuất nguyên lý và lưu vào đây. Mọi Command và Skill khi chạy đều phải đọc file này.

## [Ví dụ] Lỗi phổ biến cần tránh
- **Trạng thái:** ACTIVE
- **Phản hồi từ User:** "Đừng bao giờ mở bài bằng cụm 'Trong kỷ nguyên số ngày nay...'"
- **Bản năng được hình thành:** Mở bài đi thẳng vào vấn đề của người dùng, không dạo đầu bằng các bối cảnh vĩ mô (kỷ nguyên số, thế giới công nghệ, v.v.).

*(Các Agent có nhiệm vụ tự động append (thêm) các bài học mới vào cuối file này trong quá trình tương tác.)*
